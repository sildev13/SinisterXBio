## ภาพรวมของโปรเจกต์ (Overview)

**Young Life 2544** เป็นโปรเจกต์เกมแนว Roleplay บนแพลตฟอร์ม Roblox ที่จำลองบรรยากาศชีวิตในเรือนจำช่วงปี พ.ศ. 2544 โดยเน้นความสมจริงของระบบ Gameplay และการมีส่วนร่วมของผู้เล่นจำนวนมาก ภายใต้การดูแลของ **Team SinisterX** ผมได้รับหน้าที่เป็นหลักในการวางโครงสร้างระบบ และควบคุมทิศทางศิลป์ (Art Direction) เพื่อให้ผู้เล่นได้รับประสบการณ์ที่แปลกใหม่และสนุกสนานที่สุด

## ฟีเจอร์หลัก (Key Features)

- **ระบบอาชีพและ Roleplay**: ออกแบบระบบเศรษฐกิจภายในเกมที่สมดุล พร้อมอาชีพที่หลากหลาย ทั้งฝั่งนักโทษและเจ้าหน้าที่เรือนจำ
- **Custom UI/UX**: พัฒนาหน้าต่างอินเตอร์เฟซให้ใช้งานง่ายและเข้ากับธีมย้อนยุคของเกม เพื่อเพิ่มอรรถรสในการเล่น
- **ระบบ Admin Panel**: สร้างระบบจัดการหลังบ้านที่ทันสมัย เพื่อให้ทีมงานสามารถดูแลความสงบเรียบร้อยภายในเซิร์ฟเวอร์ได้แบบ Real-time
- **Optimized Performance**: ปรับแต่งตัวเกมให้รองรับผู้เล่นจำนวนมากพร้อมกัน (100+ Players) โดยที่ยังคงความลื่นไหลและลดอาการ Lag

## เทคโนโลยีที่ใช้ (Technologies Used)

- **Roblox Studio & Luau**: ใช้ในการเขียนสคริปต์ระบบหลักทั้งหมดของเกม
- **Blender**: สำหรับสร้าง Model 3D และสิ่งก่อสร้างที่เน้นเอกลักษณ์เฉพาะตัว
- **Web Admin Panel**: พัฒนาเว็บสำหรับจัดการฐานข้อมูลผู้เล่นโดยใช้ JavaScript และ Next.js
- **Discord API**: เชื่อมต่อระบบ Log และการแจ้งเตือนต่างๆ เข้ากับ Discord ของทีมงาน

## ความท้าทายและสิ่งที่ได้เรียนรู้ (Challenges and Learnings)

ความท้าทายที่ใหญ่ที่สุดคือการจัดการระบบฟิสิกส์และ Network ในขณะที่มีผู้เล่นรวมตัวกันจำนวนมาก ผมต้องเรียนรู้วิธีการเขียน Code ที่ประหยัดทรัพยากร (Optimization) และการใช้ Remote Events อย่างมีประสิทธิภาพ นอกจากนี้ยังได้ฝึกทักษะการบริหารจัดการทีมพัฒนา ทั้งในส่วนของ Modeler และ UI Designer เพื่อให้โปรเจกต์เสร็จทันเวลา

## ผลลัพธ์ (Outcome)

ปัจจุบัน **Young Life 2544** ได้รับการตอบรับอย่างดีเยี่ยมจากชุมชนผู้เล่น Roblox มียอดผู้เล่นพร้อมกัน (Concurrent Players) สูงสุดทะลุ 100 คน และยังคงมีการอัปเดตฟีเจอร์ใหม่ๆ อย่างต่อเนื่องทุกวัน โปรเจกต์นี้พิสูจน์ให้เห็นถึงศักยภาพของ Team SinisterX ในการสร้างสรรค์เกมที่ตอบโจทย์ผู้เล่นชาวไทยอย่างแท้จริง
